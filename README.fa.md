# 💎 کایا گالری — پلتفرم فروش طلا و جواهرات فاخر

**[English Version](README.md)** | **نسخه فارسی**

پلتفرم تجارت الکترونیکی طلا و جواهرات حرفه‌ای و آماده برای تولید، ساخته شده با معماری تمیز و مقیاس‌پذیر.

> رنگ‌های برند: `#595959` (خاکستری تیره) و `#eecdbc` (طلایی سرخ)

---

## ✨ ویژگی‌های برجسته

- **رابط کاربری و تجربه کاربر فاخر** طراحی شده برای برند لاکچری جواهرات (RTL فارسی، موبایل‌اول).
- **قیمت‌گذاری طلا به صورت لحظه‌ای** با محاسبه خودکار قیمت برای هر عیار و امکان بازنویسی توسط مدیر.
- **سبد خرید هوشمند با قفل قیمت 5 دقیقه‌ای** + نمایش شمارش معکوس + تازه‌سازی خودکار.
- **جریان کار صاف و متمرکز بر تبدیل** ورود OTP → کاتالوگ → سبد خرید → صورت‌حساب → پرداخت.
- **پنل مدیری کامل** برای محصولات، سفارشات، دسته‌بندی‌ها، کدهای تخفیف، وبلاگ، قیمت‌های طلا، کاربران، پیام‌ها و تنظیمات سایت.
- **معماری تمیز** (.NET 9) + واجهه Next.js + CRM React (Vite)، الهام‌گرفته از ساختار ثابت‌شده .

## 🚀 شروع سریع (توسعه)

### الزامات

- **.NET 9 SDK** (`dotnet --version` ≥ 9.0)
- **Node.js 20+** و **npm** (یا **yarn**)
- **SQL Server LocalDB** (ویندوز). در لینوکس/macOS، رشته اتصال را در `Api/Api/appsettings.json` تغییر دهید.

### 1. اجرای API

```powershell
cd Api/Api
dotnet restore
dotnet run
```

API خواهد:

- تمام Migration‌های EF را خودکار اعمال کند.
- داده‌های اولیه (دسته‌بندی‌ها، محصولات نمونه، پست‌های وبلاگ، اسلایدرها، نظرات، صفحات، کاربر مدیر پیش‌فرض و غیره) را Seed کند.
- در `http://localhost:5169` با Swagger UI در `http://localhost:5169/swagger` شروع شود.

**ورود مدیر پیش‌فرض (مبتنی بر OTP):**

- شماره موبایل: `09000000000`
- کد OTP: در زمان توسعه در پاسخ API بازگردانده می‌شود (همچنین در console چاپ می‌شود).

### 2. اجرای واجهه (Next.js)

```powershell
cd KaiaGalleryFront
npm install
npm run dev
```

`http://localhost:3000` را باز کنید.

### 3. اجرای پنل مدیری (React + Vite)

```powershell
cd crm-frontend
npm install
npm run dev
```

`http://localhost:5173` را باز کنید.

## 🔑 ویژگی‌های کلیدی

### واجهه فروشگاه (`/KaiaGalleryFront`)

- صفحه خانگی لاکچری با اسلایدر Hero، تیکر طلا، نمایش دسته‌بندی‌ها، پرفروش‌ترین‌ها، محصولات جدید، نظرات و پیش‌نمایش وبلاگ.
- کاتالوگ محصول فاخر با فیلترینگ غنی (جستجو، دسته‌بندی، عیار، محدوده قیمت، مرتب‌سازی).
- صفحه جزئیات محصول با گالری، تب مشخصات، قیمت واحد زنده، محصولات مرتبط.
- **سبد خرید با شمارش معکوس** (قفل قیمت 5 دقیقه‌ای + تازه‌سازی دستی + پیشنهادات محصول مشابه).
- احراز هویت مبتنی بر OTP ساده‌شده.
- صورت‌حساب با روش‌های حمل‌ونقل، کدهای تخفیف و درگاه پرداخت تقلبی.
- پروفایل مشتری، تاریخچه سفارشات با صفحه جزئیات آماده برای PDF، لیست علاقه‌مندی.
- وبلاگ با دسته‌بندی‌ها، جستجو، نمایش مقاله کامل + پست‌های مرتبط.
- صفحات درباره، تماس‌ با‌ما، و CMS پویا (`/page/[slug]`).

### پنل مدیری (`/crm-frontend`)

- **داشبورد** با KPI‌ها (درآمد، سفارشات، محصولات، مشتریان) + نمودار میله‌ای 14 روزه + نمودار دایره‌ای 5 محصول برتر.
- **مدیریت محصول** با ویرایشگر تفصیلی (وزن، عیار، دستمزد، سود، مالیات، قیمت اضافی، تصاویر، SEO، پرچم‌ها).
- **مدیریت دسته‌بندی چند‌سطحی**.
- **مدیریت سفارشات** با فیلترینگ، نمایش جزئیات، به‌روزرسانی وضعیت و کدهای پیگیری.
- **مرکز کنترل قیمت طلا** (تیکر زنده + بازنویسی دستی قیمت).
- **مدیریت کد تخفیف**.
- **ویرایشگر وبلاگ** (انتشار، نمایش بر روی صفحه اول، محتوای HTML کامل، متا SEO).
- **مدیریت کاربر** (جستجو و صفحه‌بندی مشتریان).
- **صندوق ورودی پیام مشتری** با امکان پاسخ.
- **تنظیمات سایت** (ویرایشگر کلید-مقدار برای Headers، Footers، لینک‌های شبکه‌های اجتماعی و غیره).
- دسترسی بر اساس نقش (فقط `SuperAdmin`، `SalesManager`، `WarehouseManager` می‌توانند وارد شوند).

### API (`/Api`)

- معماری تمیز: Domain → Application → Infrastructure → Api.
- ASP.NET Identity + JWT (توکن‌های دسترسی و تازه‌سازی) + ورود مبتنی بر OTP.
- EF Core 9 + SQL Server LocalDB + Migration‌های خودکار و داده‌های اولیه.
- وظیفه پس‌زمینه `GoldPriceRefreshJob` (هر 10 دقیقه) با ارائه‌دهنده تقلبی — آن را از طریق `IGoldPriceProvider` با ارائه‌دهنده واقعی جایگزین کنید.
- درگاه پرداخت تقلبی (`IPaymentGateway`) و فرستنده پیامک تقلبی (`ISmsSender`) — در محصول نهایی با ارائه‌دهندگان واقعی جایگزین کنید.
- FluentValidation، Middleware استثناء جهانی، پاکت ApiResponse ساختاری.
- Swagger / OpenAPI برای کاوش API.

## 🧩 تصمیمات معماری

- **منطق قیمت‌گذاری طلا** در `Application/Features/GoldPrice/GoldPriceCalculator.cs` متمرکز است و توسط لیست/جزئیات محصول و قفل قیمت سبد خرید استفاده می‌شود. فرمول:
  - `goldValue = weight × pricePerGramForKarat`
  - `wage     = weight × wagePerGram`
  - `profit   = (goldValue + wage) × profitPercent`
  - `tax     = (wage + profit) × taxPercent`
  - `total   = goldValue + wage + profit + tax + additionalPrice`
- **قفل قیمت سبد**: پنجره‌ای 5 دقیقه‌ای سبد را با `PriceLockExpiresAt` تمبر می‌زند. قیمت‌ها و قیمت نقطه طلا در سبد ثبت‌شده و تا زمانی که کاربر صراحتاً تازه‌سازی کند یا قفل منقضی شود پایدار می‌مانند.
- **OTP**: کدها در `OtpVerifications` ذخیره می‌شوند. در توسعه، کد به عنوان `devCode` بازگردانده می‌شود. `ISmsSender` را به یک ارائه‌دهنده واقعی متصل کنید.

## 🛠️ سفارشی‌سازی

- **رنگ‌های تم**: `KaiaGalleryFront/src/theme/palette/primary.ts`، `secondary.ts` و `crm-frontend/src/theme/index.ts` را ویرایش کنید.
- **افزودن یک منبع قیمت طلا واقعی**: `IGoldPriceProvider` را پیاده‌سازی کنید (بخش `Infrastructure/Services/GoldPrice/MockGoldPriceProvider.cs` را ببینید) و ثبت‌نام را در `Infrastructure/DependencyInjection.cs` جایگزین کنید.
- **افزودن یک ارائه‌دهنده پیامک واقعی** (مثلاً Kavenegar، Twilio): `ISmsSender` را پیاده‌سازی کنید و `DevSmsSender` را جایگزین کنید.
- **افزودن درگاه پرداخت واقعی**: `IPaymentGateway` را پیاده‌سازی کنید (حاضراً تقلبی و خودکار موفق).
