## 🇮🇷 فارسی

ما از مشارکت‌های شما در Kaia Gallery استقبال می‌کنیم! برای کمک، لطفاً این راهنما را دنبال کنید.

### مراحل شروع

1. **Fork** این Repository را Fork کنید
2. **Clone** پروژه را Clone کنید: `git clone https://github.com/YOUR_USERNAME/kaiagoldgallery.git`
3. **Branch** یک Branch جدید بسازید: `git checkout -b feature/your-feature`
4. **Commit** تغییرات خود را Commit کنید: `git commit -m "feat: توضیح ویژگی"`
5. **Push** تغییرات را Push کنید: `git push origin feature/your-feature`
6. **Pull Request** یک Pull Request بسازید

### استانداردهای کد

- **کد پاک**: از کد خوانا و تمیز استفاده کنید
- **تست**: قبل از ارسال تغییرات، تست‌های خود را بنویسید
- **Documentation**: کدهای جدید را مستند کنید
- **Commit Messages**: پیام‌های Commit را واضح و توصیفی بنویسید

### سوالات و مشکلات

اگر سوالی دارید:
- **Issues**: یک Issue بسازید
- **Discussions**: بحث را شروع کنید

## 🇬🇧 English

We welcome contributions to Kaia Gallery! Please follow this guide to get started.

### Getting Started

1. **Fork** this repository
2. **Clone** the project: `git clone https://github.com/YOUR_USERNAME/kaiagoldgallery.git`
3. **Create a Branch**: `git checkout -b feature/your-feature`
4. **Commit** your changes: `git commit -m "feat: description of your feature"`
5. **Push** your changes: `git push origin feature/your-feature`
6. **Create a Pull Request**

### Code Standards

- **Clean Code**: Write readable and maintainable code
- **Tests**: Write tests for your changes
- **Documentation**: Document new code
- **Commit Messages**: Write clear and descriptive commit messages

### Questions & Issues

If you have questions:
- **Create an Issue**: Report bugs or request features
- **Start a Discussion**: Ask general questions

---

## 📋 Pull Request Checklist

- [ ] My code follows the style guidelines
- [ ] I have performed a self-review of my own code
- [ ] I have commented my code, particularly in complex areas
- [ ] I have made corresponding changes to the documentation
- [ ] My changes generate no new warnings
- [ ] I have tested my changes locally
