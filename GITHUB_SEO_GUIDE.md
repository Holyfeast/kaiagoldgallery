### نحوه اضافه کردن Topics

1. اپن Repository خود را روی GitHub
2. Gear icon (⚙️) در بخش About را کلیک کنید
3. "Topics" را پیدا کنید
4. تک تک کلمات کلیدی بالا را اضافه کنید

### How to Add Topics

1. Open your Repository on GitHub
2. Click the Gear icon (⚙️) in the About section
3. Find "Topics"
4. Add each keyword above one by one

### Recommended Labels

Use these labels when creating Issues and PRs:

- `bug` - Bug reports
- `enhancement` - New features
- `documentation` - Docs updates
- `good first issue` - For newbies
- `help wanted` - Need contributors
- `question` - Questions
- `backend` - API issues
- `frontend` - UI/Frontend issues
- `security` - Security vulnerabilities
